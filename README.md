# Phishing Awareness Training Module

A cybersecurity awareness presentation/module designed to educate users about phishing attacks, social engineering tactics, and online safety practices.

## Project Overview

This project aims to increase awareness about phishing threats by helping users identify suspicious emails, fake websites, and manipulation techniques used by cybercriminals. The module also includes practical examples and interactive quizzes for better engagement and learning.

## Features

- Understanding phishing attacks and their impact
- Identifying phishing emails
- Recognizing fake websites
- Learning common social engineering tactics
- Best practices to avoid phishing scams
- Real-world phishing examples
- Interactive quizzes and scenario-based activities

## Learning Objectives

After completing this module, users will be able to:

- Explain what phishing attacks are
- Detect common phishing indicators
- Identify fake websites and suspicious URLs
- Understand attacker manipulation techniques
- Apply security best practices
- Respond appropriately to suspected phishing attempts

## Topics Covered

### What is Phishing?
- Definition and types of phishing
- Why phishing attacks are dangerous

### Recognizing Phishing Emails
- Suspicious sender addresses
- Urgent messages and threats
- Fake attachments and links
- Requests for sensitive information

### Detecting Fake Websites
- Misspelled domains
- HTTPS verification
- Unrealistic offers
- Website red flags

### Social Engineering Tactics
- Authority-based attacks
- Fear and urgency tactics
- Curiosity bait
- Reward-based scams

### Best Practices

✔ Verify sender information  
✔ Avoid clicking suspicious links  
✔ Enable Multi-Factor Authentication (MFA)  
✔ Use strong passwords  
✔ Keep systems updated  
✔ Report suspicious activity  

## Real-World Examples

- Business Email Compromise (BEC) scams
- COVID-19 phishing campaigns
- Credential theft attacks

## Interactive Learning

- Phishing identification exercises
